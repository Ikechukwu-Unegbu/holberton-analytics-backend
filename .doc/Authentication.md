
## Register New User
    Make a post request to this endpoint
    url: 'https://domainname/register',
    Method: 'POST',
    Headers: 'Accept: application/json',
    Body:
    {
        "fullname":"Lucia Chinenye", 
        "username":"Lucia",
        "email":"Lucia@gmail.com",
        "password":"password"
    }
